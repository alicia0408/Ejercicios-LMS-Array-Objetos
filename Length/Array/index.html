getNthElement

Escriba una función llamada getNthElement.

Dado un arreglo y un entero, getNthElement devuelve el valor según el entero dado, dentro del arreglo dado.

Notas: Si el arreglo tiene una longitud de 0, debería devolver' undefined'. Ejemplo de entrada: var output = getNthElement([1, 3, 5], 1); console.log(output); // --> 3
